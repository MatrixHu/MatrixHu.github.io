HEXO新建文章

1.资源管理器进入站点文件夹
C:\Users\Kevin\blog\blog

2.右键Git Bash Here

3.创建新文章
$ hexo new “title”
完成后，会在 source/_post 文件夹下创建一个md文件，命名为：title.md
这个文件就是将要发布到网站上的原始文件，用于记录文章内容

4.编写文章
编辑器打开.md文件

5.部署文件
$ hexo clean
$ hexo g

6.本地测试
$ hexo s
可在浏览器 localhost:4000 预览

7.发布到网络
$ hexo d
